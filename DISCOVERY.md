The latent value of `organvm/client-sdk` lies in its ability to serve as a high-performance, real-time WebSocket communication layer for generative performance infrastructure. By treating audience interactions not as stateless API requests but as continuous, real-time feedback loops over typed WebSocket channels, it provides a foundational mechanism for participatory art. Its highest concrete value is as a decoupled real-time synchronization library that can be consumed by external applications or frontend UIs to enable multiplayer generative performance experiences, abstracting away complex protocol mechanics like reconnection, latency compensation, and topic-based sub/pub. The single best concrete first task to unlock this value is to implement the core WebSocket transport layer and Protocol Buffer / MessagePack codec for efficient bidirectional communication.
